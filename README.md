## 使用

### 安装

	git clone git@115.28.80.125:skillee/skilleeweb.git SkilleeWeb
	cd SkilleeWeb
	npm install

### 启动开发环境

	npm run dev

然后打开浏览器访问: http://localhost:8080


### 编码打包生成代码

	npm run build

### 代码规范检查

	npm run lint
